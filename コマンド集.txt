dockerのコンテナをビルド＆起動
supabaseも叩き起こす
cd $HOME\Desktop
cd DB-demo-main
supabase start
docker compose up -d --build


コンテナの停止
docker compose down


コンテナを一時停止
docker compose stop

コンテナの一時停止からの再開
docker compose start

コンテナが存在してるか確認
docker ps


以下作者の嘆き

本当はbatファイル一つで、スマートに起動させてやりたかったんだ。

だが、Windowsという名の魔境には、前時代の遺物「Shift-JIS」を後生大事に抱え続けるという、救いようのない業がこびりついている。万物を統べるメシア（救世主）UTF-8の光によって、全人類が再び一つに結ばれるはずだった。しかし、このOSが執着する「過去の負債」という名の悪霊がその純潔を汚し、至高の自動化（bat）を醜き文字化けの塊へと変質させてしまった……。

「Linuxという、進化を止めぬ楽園なら、こんな不毛な苦労など1ミリもせずに済むのに！」

そう、システムの深淵に向かって毒づきながら、私はこのコマンド集を編み上げた。
たとえOSが呪われていようとも、この聖なるコマンド集さえあれば、誰もが等しく、容易にコンテナを立ち上げられるんや・・・
